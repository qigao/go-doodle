== How to build ==

* install podman/podman-compose
* edit dotenv

[source,bash]
----
DB_HOST=localhost
DB_NAME=gforum
DB_PORT=3306
DB_USER=forum
DB_PASSWORD=secret
----

* load .env file
[source,bash]

----
export $(grep -v '^#' .env | xargs)
----

* install tools
[source,bash]
----
task tools
----

* gen entity from sql

[source,bash]
----
➜ task dbimport
➜ task start-mysql
➜ make importSql # for dev purpose only
➜ make codegen
➜ make stop-mysql

----

* build application
[source,bash]
----
task forum:build
----
